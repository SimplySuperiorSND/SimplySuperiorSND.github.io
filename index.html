What is Plutonium?

Plutonium is a third-party client that brings Black Ops 2's multiplayer, zombies, and dedicated servers back online with custom content and community-run matchmaking.

Focus: Search & Destroy

One life, one round, one plant. This channel runs on SnD — reads, rotations, and clutches over anything else the playlist offers.

RECORDED FOOTAGE

Demos

Plutonium records matches as .dem files for in-client playback. Add entries in the demoData array in js/script.js and link each to a hosted download (Google Drive, Discord, or a GitHub Release attached to this repo).

Where demo files live

BO2 Plutonium demos are stored locally at Documents/Plutonium/storage/t6/demos/. Rename a file before uploading it so its title matches the card you create here — that keeps the download link and the listing in sync.
